Rollup.js配置

2021年8月3日

文章目录

  1. Rollup + TS
  2. 参考

rollup.js JavaScript 模块打包器,相比webpack配置更简单轻量。适合library使用的打包工具。

它的特点：

  1. 只想快速配置开发个小工具
  2. 要用ES6+语法开发
  3. 代码清洁，清理无用代码(tree-shaking)

它输出各种模块规范：AMD、CommonJS、UMD和IIFE 。
